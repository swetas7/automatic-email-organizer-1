# Automatic Email Organizer using NLP

This project is a hackathon prototype that automatically reads unread Gmail emails,
classifies them using zero-shot NLP models, applies labels, and displays the results
in a simple dashboard.

## Features
- Gmail API integration
- Automatic email classification
- Priority detection
- Explainable AI decisions
- Clean dashboard view

## Tech Stack
- Python
- Gmail API
- Hugging Face Transformers
- Streamlit
- SQLite
